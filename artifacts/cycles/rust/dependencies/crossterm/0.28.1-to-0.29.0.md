## Summary

Despite the version bump appearing major, this is a **semver-minor update** with no breaking API changes. Crossterm uses 0.x versioning where minor bumps can include new features.

## Added Features

### 1. OSC52 Clipboard Support (#974)
Copy to clipboard using OSC52 escape sequences.

**New API:**
```rust
// New clipboard functionality
use crossterm::clipboard;
```

### 2. Enhanced KeyboardEnhancement Queries (#958)
Added `query_keyboard_enhancement_flags()` to read currently enabled flags.

### 3. Trait Derivations (#909)
Standard traits derived for `SetCursorStyle`:
- `Debug`, `Clone`, `Copy`, `PartialEq`, `Eq`, etc.

### 4. Event Enum Helpers (#949)
Added `is_*` and `as_*` methods to event enums:

```rust
if event.is_key() {
    let key = event.as_key().unwrap();
}
```

### 5. Feature Flag for derive_more (#970)
New feature flag to control `derive_more` implementations.

## Dependency Updates

- **rustix:** Updated to 1.0 (#982)
- Various other minor dependency upgrades

## Bug Fixes

- **KeyModifiers Display:** Fixed to properly add `+` between modifiers (#979)

## Context: 0.28 Breaking Changes

For reference, **0.28** had breaking changes:
- Switched to Rustix by default (libc available via feature flag)
- Fixed duplicate bit masks for caps/num lock (breaks `KeyEventState` serialization)

**If you're on 0.28.1, these are already handled.**
