# Palindrome

**Topic:** linguistics
**Scope:** global
**Date:** 2025-11-30

## Knowledge

A **palindrome** is a word, phrase, or sequence that reads the same forwards and backwards.

Examples:
- Words: racecar, level, kayak, civic
- Phrases: "A man, a plan, a canal: Panama"
- Numbers: 12321

Related terms:
- **Semordnilap**: word that spells a different word backwards (stressed/desserts)
- **Emordnilap**: same thing, different name

## Why It Matters

- Classic programming interview question (check if string is palindrome)
- Fun linguistic category
- Some languages have more natural palindromes than others

## Quiz Question

What's the term for a word that's spelled the same forwards and backwards?

## Answer

Palindrome.
