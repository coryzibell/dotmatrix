# DOM Acronym

**Topic:** web
**Scope:** global
**Date:** 2025-11-30

## Knowledge

**DOM** = **Document Object Model**

The tree structure browsers build from HTML. JavaScript interacts with the page by manipulating this tree (adding nodes, changing attributes, etc.).

Related:
- Virtual DOM: React's optimization layer that diffs before touching real DOM
- Shadow DOM: encapsulated DOM trees for web components
- CSSOM: CSS Object Model (parallel tree for styles)

## Why It Matters

- Foundation of all web interactivity
- DOM manipulation is slow - why frameworks batch updates
- Understanding the DOM tree helps debug layout issues
