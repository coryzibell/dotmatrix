# HTTP 418 I'm a Teapot

**Topic:** http
**Scope:** global
**Date:** 2025-11-30

## Knowledge

**418 I'm a teapot** - an April Fools' joke from RFC 2324 (1998), the Hyper Text Coffee Pot Control Protocol (HTCPCP). A teapot should return 418 if asked to brew coffee.

Despite being a joke, it's been implemented in real servers (Google, etc.) and nearly got removed from Node.js in 2017, sparking controversy.

Nearby codes:
- 422 Unprocessable Entity - valid syntax, semantic error
- 451 Unavailable For Legal Reasons - named after Fahrenheit 451

## Why It Matters

- Shows the humor in internet standards
- Actually used as an Easter egg in production systems
- Demonstrates that RFCs aren't always serious

## Quiz Question

What HTTP status code means "I'm a teapot"?

## Answer

418.
