# CatDog - Which End is Which

**Topic:** cartoons
**Scope:** global
**Date:** 2025-11-30

## Knowledge

In *CatDog* (Nickelodeon, 1998-2005):
- **Cat** is on the front/right (walks forward) - smart, scheming, cultured
- **Dog** is on the back/left (walks backward) - dumb, friendly, loves garbage trucks

They're conjoined brothers with no explanation given for their existence. The show never addressed how they... you know.

Created by Peter Hannan.

## Quiz Question

In CatDog, which end is the cat and which end is the dog?

## Answer

Cat is front/right, Dog is back/left.
