# Alaska: The Home Row State

**Topic:** trivia
**Scope:** global
**Date:** 2025-11-30

## Knowledge

**Alaska** is the only US state that can be typed using only one row of a standard QWERTY keyboard - specifically the home row (ASDFGHJKL).

A-L-A-S-K-A: all home row keys.

## Why It Matters

- Fun keyboard/typing trivia
- Good pair with "Q is the only letter not in any state name"
- Other single-row words: "typewriter" (top row), "alfalfa" (home row)

## Quiz Question

What's the only US state that can be typed on a single row of a standard QWERTY keyboard?

## Answer

Alaska (home row: ASDFGHJKL).
