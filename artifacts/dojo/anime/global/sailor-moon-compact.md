# Sailor Moon - Transformation Compact

**Topic:** anime (magical girl / Filipino era)
**Scope:** global
**Date:** 2025-11-30

## Knowledge

**The Transformation Brooch** (Moon Prism Compact) is Usagi's first transformation item in Sailor Moon.

**Details:**
- A circular makeup compact/brooch
- Features the crescent moon symbol
- Given to her by Luna
- Used with the phrase "Moon Prism Power, Make Up!"

**Later transformation items:**
- Crystal Star (Season 2) - "Moon Crystal Power, Make Up!"
- Cosmic Heart Compact (Season 3) - "Moon Cosmic Power, Make Up!"
- Crisis Moon Compact (Season 4) - "Moon Crisis, Make Up!"
- Eternal Moon Article (Season 5) - "Moon Eternal, Make Up!"

The compact became iconic - every magical girl show since has some transformation trinket.

## Quiz Question

In Sailor Moon, what item does Usagi use to transform into Sailor Moon in the first season?

## Answer

The Transformation Brooch / Moon Prism Compact (a makeup compact).
