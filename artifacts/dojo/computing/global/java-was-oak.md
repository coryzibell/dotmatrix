# Java Was Originally Called Oak

**Topic:** computing
**Scope:** global
**Date:** 2025-11-30

## Knowledge

**Java** was originally called **Oak**, named after an oak tree outside James Gosling's office at Sun Microsystems in 1991.

Renamed to Java in 1995 because "Oak" was already trademarked by Oak Technologies. The name Java came from Java coffee - the team was brainstorming at a coffee shop.

## Why It Matters

- Classic programming history trivia
- Other renamed languages: C++ was "C with Classes", JavaScript was briefly "Mocha" then "LiveScript"
- Shows how trademark issues shape tech history
