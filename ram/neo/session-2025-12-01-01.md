### Skipper - Copilot CLI Config
New project at `~/.skipper/` - lean config for Copilot CLI (corporate work).

**Structure:**
```
~/.skipper/
├── setup.py                     # Syncs to ~/.github/ and ~/.copilot/
├── copilot-instructions.md      # Skipper identity - orchestrator
├── agents/
│   ├── _base.md                 # Shared agent defaults
│   ├── builder.agent.md         # Write code
│   ├── scout.agent.md           # Explore/research
│   ├── critic.agent.md          # Review
│   └── doc.agent.md             # Documentation
└── memory/
    └── {agent}/{short,long}/    # Filesystem-based memory
```

**Key decisions:**
- No MCP memory server - filesystem short/long memory instead
- Skipper orchestrates, agents execute (like Neo/Matrix but simpler)
- Four focused agents vs full Matrix identity roster
- `setup.py` syncs instructions → `~/.github/copilot-instructions.md`, agents → `~/.copilot/agents/`
